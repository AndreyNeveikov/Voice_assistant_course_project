Голосовой ассистент

Вы можете разговаривать с ассистентом на общие темы - про его возраст, пол, кто его создал, как у него дела и так далее.

Помимо общих тем вы можете дать асситенту следующие команды:

- Найди + любой поисковый запрос
- Смотреть на ютуб + любой поисковый запрос
- Слушать песню + название песни
- Расскажи анекдот
- Запустить стандартное приложение:
     - калькулятор
     - paint
     - блокнот
     - браузер
- Запустить установленное приложение:
     - Telegram
     - Торрент
     - Sublime text
     - Skype
     - Google
     - GitHub Desktop
- Ответь + ваш вопрос; Наприер:
     - Кто тебя создал
     - Что такое паттерн проектирования?
     - Что ты умеешь
     - Как настроение
     - Какие библиотеки используешь
