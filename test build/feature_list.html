Голосовой ассистент

Вы можете дать асситенту следующие команды:

- Найди + любой поисковый запрос
- Смотреть на ютуб + поисковый запрос
- Слушать песню + название песни
- Расскажи анекдот
- Запустить приложение:
     - калькулятор
     - paint
     - блокнот
     - браузер
     - Telegram
     - Торрент
     - Sublime text
     - Skype
     - Google
     - GitHub Desktop
- Ответь + ваш вопрос; Наприер:
     - Кто тебя создал?
     - Что такое паттерн проектирования?
     - Что такое ООП?
     - Как тебя зовут?
     - Какие библиотеки используешь?
- База данных + команда; Наприер:
     - Очисти базу
     - Добавь сайт
     - Удали запись
     - Напомни пароль/логин
